# PezeshkaneSalamat – Doctor Directory Web Application

**PezeshkaneSalamat** is a web application designed to introduce and list medical professionals across different specialties.

The platform allows users to browse doctors, view their profiles, and access contact information to help them find the right medical specialist.

This project represents a **medical directory platform** built with ASP.NET Web Forms and SQL Server.

---

https://pezeshkanesalamat.ir/

# Features

## Public Website

* Browse doctors by specialty
* View doctor profiles
* Medical category listings
* Contact information for doctors
* Simple and accessible user interface
* Responsive layout for different screen sizes

# Installation

## 1. Clone the repository

```bash
git clone https://github.com/ghaseminaeeme/PezeshkaneSalamat_wf.git
cd PezeshkaneSalamat_wf
```

---

## 2. Open the project

Open the solution file in **Visual Studio**.

---

## 3. Configure the database

Create a **SQL Server database** and update the connection string inside:

```
Web.config
```

Replace the database credentials with your own local configuration.

---

## 4. Restore dependencies

Restore NuGet packages if required.

---

## 5. Run the project

Run the project using **Visual Studio (IIS Express)**.

The application will be available on your local development server.
